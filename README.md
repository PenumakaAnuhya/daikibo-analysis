# daikibo-analysis
# 📊 Daikibo Industrial Analytics Project

## 🔍 Project Overview

This project analyzes machine telemetry data and employee compensation data for Daikibo Industrials to identify:

* Factory-level machine downtime
* Device types causing maximum failures
* Gender pay equality across roles and locations

---

## 🛠️ Tools Used

* Tableau (Data Visualization)
* Microsoft Excel (Data Processing)
* GitHub (Version Control)

---

## 📁 Dataset Description

### 1. Telemetry Data

* Machine status data collected every 10 minutes
* Includes factory location, device type, and status

### 2. Equality Data

* Factory
* Job Role
* Equality Score (-100 to +100)

---

## ⚙️ Key Steps Performed

### 📌 Task 1: Machine Downtime Analysis

* Created calculated field: `Unhealthy = 10 mins downtime`
* Built bar chart for downtime per factory
* Identified factory with highest machine failures
* Analyzed device types contributing to downtime

---

### 📌 Task 2: Gender Pay Equality Analysis

* Classified equality scores into:

  * Fair (±10)
  * Unfair (±11 to ±20)
  * Highly Discriminative (>±20)
* Processed data using Excel formulas

---

## 📊 Key Insights

* Identified the factory with maximum downtime
* Determined high-risk machine types
* Highlighted roles with potential gender pay inequality
